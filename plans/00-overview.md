# Polls App — Обзор проекта

## Цель
Веб-приложение, позволяющее пользователям создавать и проходить опросы, с аналитикой и управлением доступом.

## Технологический стек

| Слой | Технология |
|---|---|
| Бэкенд | Node.js + NestJS |
| Фронтенд | React + TypeScript + Vite |
| База данных | PostgreSQL |
| ORM | Prisma |
| Аутентификация | JWT (access + refresh токены) |
| Графики | Recharts |
| Стилизация | Tailwind CSS |
| Документация API | Swagger (OpenAPI 3.0) |
| Контейнеризация | Docker + Docker Compose |

## Функциональный охват

### Аутентификация
- Регистрация / Вход / Выход
- JWT access-токен (15 мин) + refresh-токен (7 дней)
- Защищённые маршруты

### Опросы
- Создание опроса с несколькими типами вопросов (одиночный выбор, множественный выбор, текст)
- Настройка видимости: публичный / приватный (только по ссылке)
- Установка дедлайна (expires_at)
- Редактирование / удаление своих опросов
- Поделиться опросом через уникальный slug/ссылку

### Ответы
- Прохождение опроса (анонимно или авторизованно)
- Один ответ на пользователя на опрос (дедупликация)
- Просмотр своих отправленных ответов

### Аналитика
- Количество ответов по времени (линейный график)
- Распределение ответов по вопросам (столбчатый/круговой график)
- Экспорт результатов в CSV

### Роли
- `user` — может создавать опросы и проходить их
- `admin` — может управлять всеми опросами и пользователями
